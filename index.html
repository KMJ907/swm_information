스윗미 소개

스윗미는 학생들의 공부 습관 형성을 돕기 위해 만들어진 학습 앱입니다.
공부 타이머, 노트, 학습 기록 등 학생 여러분의 공부를 도와주는 기능을 제공합니다.

주요 기능

  • 공부 시간을 기록할 수 있는 타이머 기능
  • 공부 내용을 정리할 수 있는 노트 기능
  • 학습 습관 형성을 도와주는 성장형 기능
  • 학생들이 더 꾸준히 공부할 수 있도록 돕는 편의 기능

APK 설치 방법

  1. APK 다운로드 버튼을 눌러 APK 파일을 다운로드합니다.
  2. 다운로드가 완료되면 APK 파일을 실행합니다.
  3. “알 수 없는 앱 설치” 안내가 표시되면 설정 화면으로 이동합니다.
  4. Chrome 또는 파일 관리자에서 “이 출처 허용”을 활성화합니다.
  5. 다시 APK 파일을 실행하여 설치를 진행합니다.

Family Link가 설정된 경우

Family Link가 적용된 기기에서는 APK 설치가 제한될 수 있습니다.

  1. 보호자 기기에서 Family Link 앱을 실행합니다.
  2. 자녀 계정을 선택합니다.
  3. 제어 또는 설정 메뉴로 이동합니다.
  4. 자녀 기기 설정을 확인합니다.
  5. “알 수 없는 앱 설치” 항목이 있다면 허용합니다.
